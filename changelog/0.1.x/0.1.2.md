# 0.1.2 — 2026-05-23

## Fixed

- **`gbif_search_publishers`** — endpoint path corrected from `/organization/search` to `/organization`; the wrong path returned 404s on every call.
- **`gbif_get_occurrence`, `gbif_search_occurrences`** — occurrence heading now falls back to `scientificName` when `canonicalName` is absent, replacing the misleading "Unknown taxon" label.
- **`gbif_get_dataset`, `gbif_search_datasets`** — dataset `description` fields are now HTML-stripped before surfacing; raw markup and HTML entities no longer leak into agent-facing output.
- **`gbif_get_species_children`** — removed fabricated `totalCount` field; the GBIF children endpoint does not return a total count and the prior value (`raw.count ?? 0`) was always `0`.
