# 0.1.0 — 2026-05-23

## Added

- **`gbif_match_species`** — match a scientific or common name against the GBIF backbone taxonomy; returns `taxonKey`, confidence score (0–100), full classification, and `matchType` (EXACT, FUZZY, HIGHERORDER, NONE).
- **`gbif_get_species`** — fetch a complete taxon record by backbone key; includes authorship, vernacular name, synonymy (`acceptedKey`/`accepted`), descendant/occurrence counts, `publishedIn`, and optional `extinct` flag.
- **`gbif_search_species`** — search or browse the backbone by name fragment, rank, kingdom, family, or genus; filter to extinct/extant taxa or a specific checklist dataset.
- **`gbif_get_species_classification`** — return the complete parent chain (kingdom → immediate parent) as an ordered array with rank, name, and key per node.
- **`gbif_get_species_children`** — list direct children of a backbone taxon (genera within a family, species within a genus, etc.); paginated up to 1000.
- **`gbif_search_occurrences`** — search 2.4B+ GBIF occurrence records with Darwin Core filters: `taxonKey`, country (ISO 3166-1 alpha-2), bounding box, WKT polygon, year range, month, `basisOfRecord`, `hasCoordinate`, `isInCluster`; pagination capped with detection at ≈100,000 offset+limit.
- **`gbif_count_occurrences`** — fast single-number count via `/occurrence/count`; supports `taxonKey`, `country`, `isGeoreferenced`, `datasetKey`, `year`.
- **`gbif_get_occurrence`** — fetch a single Darwin Core occurrence record by key; full coordinate, GADM geography, collections metadata, collector/identifier names, media links, and quality issue flags.
- **`gbif_occurrence_facets`** — aggregate occurrence counts by dimension (COUNTRY, YEAR, BASIS_OF_RECORD, DATASET_KEY, kingdom/phylum/class/order/family/genus/species keys, PUBLISHING_COUNTRY, MONTH); returns top-N values ranked by count.
- **`gbif_search_datasets`** — search GBIF datasets by keyword, type, publishing country, or hosting organization; returns title, type, description, license, DOI, record count.
- **`gbif_get_dataset`** — fetch full dataset metadata by UUID; includes citation text, contacts, `numConstituents`, license, DOI.
- **`gbif_search_publishers`** — search GBIF-registered organizations by name or country; keys chain into `gbif_search_datasets` `hostingOrg`.
- **`gbif://species/{taxonKey}`** resource — stable URI for backbone taxon records (classification, authorship, synonymy, vernacular name).
- **`gbif://dataset/{datasetKey}`** resource — stable URI for dataset metadata (title, description, citation, license, coverage).
