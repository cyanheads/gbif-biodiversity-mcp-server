# 0.2.8 — 2026-06-15

## Added

- **`createApp({ instructions })`** — server-level orientation in `src/index.ts`, sent to clients on every `initialize`. Steers callers to `gbif_match_species` for backbone `taxonKey` resolution, notes the keyless API, ISO 3166-1 alpha-2 countries, UUID dataset/publisher keys, and the occurrence paging cap at offset+limit ≈ 100,000.

## Changed

- **Plugin display identity** unscoped to the bare package name `gbif-biodiversity-mcp-server` — top-level `name` in `.claude-plugin/plugin.json` and `.codex-plugin/plugin.json`, the `mcpServers` key in `.claude-plugin/plugin.json`, and the server key in `.codex-plugin/mcp.json`. The npm install `args` stay scoped (`@cyanheads/gbif-biodiversity-mcp-server`).
